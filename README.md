# 강영민 · 전산직 포트폴리오

전산직 취업을 위한 개인 역량 소개 및 기술 포트폴리오 페이지입니다.

**한 줄 소개:** Java와 DB 연동 기반으로 직접 부딪히며 문제를 해결해온 실전형 전산 인재

---

## 기술 스택

- **HTML5** – 시맨틱 마크업, 단일 페이지 구조
- **CSS3** – 반응형 레이아웃, CSS 변수, 애니메이션
- **JavaScript (Vanilla)** – 스무스 스크롤, 모바일 메뉴, 스크롤 등장 효과

배포는 **GitHub Pages** 기준으로 구성되어 있습니다.

## 로컬에서 보기

1. 저장소 클론 후 프로젝트 폴더로 이동  
2. `index.html`을 브라우저에서 열기  

또는 로컬 서버 사용:

```bash
# Python 3
python -m http.server 8000

# Node (npx)
npx serve .
```

이후 브라우저에서 `http://localhost:8000` 접속

---

## GitHub Pages 배포

1. 이 저장소를 GitHub에 푸시
2. **Settings** → **Pages**
3. **Source**: `Deploy from a branch`
4. **Branch**: `main` (또는 사용 중인 기본 브랜치), `/ (root)` 선택
5. 저장 후 잠시 기다리면 `https://<username>.github.io/myself/` 주소로 접속 가능
